Package: seq_model
=======================
Classes for modeling proteins with support for .pdb i/o

Chain:
  Class that models polypeptide chains, which are sequences of Residue objects.

  
Residue:
  Class that models amino acid residues.  Residues aggregate Atom objects.


Secel:
  Class that models secondary structure elements.
  subclassed by Helix, Strand, Sheet, and Coil
