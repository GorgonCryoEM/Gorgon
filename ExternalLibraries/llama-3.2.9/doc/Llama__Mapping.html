Detailed Description

Mapping class. Implements a mapping from R^N x R^P -> R^N

 Mapping (void)

Mapping

 Mapping (const std::string& what, Index N, Index P, typename Diffeomorphism::diffeo_fcn f, typename Diffeomorphism::diffeo_jac j = 0, Index iter = 1)

Mapping

Construct a new mapping. The arguments are: 1) a string descriptive of the mapping, 2) the dimension of the state space, 3) the dimension of the parameter space, 4) the routine which implements the mapping, 5) (optional) a routine which calculates the Jacobian of the mapping, 6) (optional) the number of times the diffeomorphism is to be iterated (default = 1).
